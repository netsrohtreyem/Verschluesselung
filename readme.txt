Hinweis zur Nutzung der RSA Verschlüsselung:

Key Datei:
privateKey.key

Enthält den private + den public Key dient zu entschlüsseln von Dateien, !nicht! weitergeben !


publicKey.key 

Enthält nur den public Key, dient zur Weitergabe an externe zum Verschlüsseln von Dateien

Ablauf Verschlüsselung:

1. Wenn nicht vorhanden public Key besorgen bzw. erzeugen

2. Datei verschlüsseln


Ablauf Entschlüsselung:

1. Mit dem eigenen private /public Key Paar (Datei) 

2. die verschlüsselte Datei entschlüsseln
